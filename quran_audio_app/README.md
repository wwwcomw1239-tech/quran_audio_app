# quran_audio_app

تطبيق القرآن الكريم بصوت الشيخ محمد صديق المنشاوي

## المميزات

- 📖 قائمة كاملة بسور القرآن الكريم
- 🎧 استماع لتلاوات الشيخ المنشاوي
- ⬇️ تحميل السور للاستماع offline
- 🌙 دعم الوضع الداكن
- 🔍 البحث في السور

## المتطلبات

- Flutter SDK >= 3.22.0
- Dart SDK >= 3.4.0

## التثبيت

```bash
flutter pub get
flutter run
```

## البناء

```bash
flutter build apk --release
```
